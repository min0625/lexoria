Lexoria - third-party license texts

Full license texts for the third-party data shipped with this app (design doc
section 14). The level JSON files embed WordNet definitions and ECDICT
translations; both licenses require the notice to accompany every copy, so this
file is distributed alongside the game.

Lexoria's own code is Apache-2.0 (see LICENSE in the repository).
The ENABLE word list is public domain and carries no notice requirement.
Build-time-only tools (OpenCC, NLTK, wordfreq) ship no data with the app and
carry no notice requirement here.
